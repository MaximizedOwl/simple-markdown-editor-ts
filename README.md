## 概要

左側で Markdown 記法で入力した内容を同期的に変換し右側に表示しながら書ける Web 上のエディターです。

## 使える文法

ライブラリ「[react-markdown](https://github.com/remarkjs/react-markdown)」に準拠します。

## シンタックスハイライトで対応している言語

ライブラリ「[react-syntax-highlight](https://github.com/react-syntax-highlighter/react-syntax-highlighter)」に準拠します。
